# Used Car Price Data Analysis

## Description

This project analyzes used car data using SQL to understand pricing patterns and factors affecting car prices such as fuel type, kilometers driven, and car age.

## Tools Used

* MySQL

## Analysis Performed

* Average Price by Fuel Type
* Price vs Kilometers Driven
* Price Distribution by Car Age
* Count of Cars by Brand and City

## Key Insights

* Diesel cars have higher average prices compared to petrol cars
* Cars with lower kilometers driven tend to have higher prices
* Newer cars have better resale value than older cars
* Certain cities show higher average prices due to demand

## Conclusion

Car price is mainly influenced by mileage, fuel type, and age of the vehicle. Lower usage and newer cars provide better value in the resale market.


## Recommendations
- Buyers should prefer cars with lower kilometers driven
- Diesel cars are better for resale value
- Sellers should maintain cars properly to get higher prices
